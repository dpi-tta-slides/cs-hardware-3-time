# CS Hardware Day 3 - Time

In this lesson, participants explore how electronic circuits interact with time using capacitors. Building on concepts from previous lessons about voltage, resistance, LEDs, and transistors, participants investigate how capacitors charge and discharge to create delays, fades, and oscillations. Through hands-on RC timing circuits and a transistor-based astable multivibrator, participants connect timing behavior to real-world computing systems such as CPU clocks, DRAM memory, and signal generation. The lesson emphasizes experimentation, debugging, measurement, and the role of timing in modern hardware systems.

[Slides](https://dpi-tta-slides.github.io/cs-hardware-3-time/)

## Learning Objectives

By the end of this lesson, participants will be able to:

- Explain how capacitors store and release electrical energy over time
- Describe charging and discharging behavior in resistor capacitor (RC) circuits
- Build and analyze resistor capacitor (RC) circuits
- Measure how resistance and capacitance affect timing
- Construct and troubleshoot an astable multivibrator oscillator
- Explain how timing circuits relate to clocks, memory, and modern computer hardware
